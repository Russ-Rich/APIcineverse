API Endpoints

Dive right into a seamless journey navigating through a treasure trove of movie databases. You can look forward to receiving JSON responses, interacting via standard HTTP protocols, and experiencing hassle-free authentication procedures.

Movie Endpoints

HTTP Method Endpoint Usage Returned Data
GET /movies Retrieve a list of all movies JSON array of all movies
GET /movies/:title Retrieve detailed information of a specific movie JSON object with movie details
GET /genres/:name Retrieve details of a specific genre JSON object with genre description
GET /directors/:name Retrieve details of a specific director JSON object with director details

User Endpoints

HTTP Method Endpoint Usage Returned Data
POST /users Register a new user JSON object confirming registration
PUT /users/:username Update details of a specific user JSON object confirming update
POST /users/:username/favorites Add a movie to a user's favorites list Confirmation message
DELETE /users/:username/favorites/:movieID Remove a movie from a user's favorites list Confirmation message or 404 error message
DELETE /users/:username Delete a user's account Confirmation message or 404 error message
